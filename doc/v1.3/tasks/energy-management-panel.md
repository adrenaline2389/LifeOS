# 能量管理系统页面任务

## 目标

实现能量管理系统的三个面板：当前能量状态、今日能量轨迹、能量风向标。

## 最小可执行任务

用户可以进入能量管理系统，选择性记录某一个维度，并看到当前状态、今日轨迹和能量风向标更新。

## 输入

- 能量维度配置。
- 能量观察点存储函数。
- 能量聚合函数。
- 复古 UI 系统组件。
- `doc/v1.3/design.md`

## 输出

- 能量管理系统页面 UI。
- 选择性记录交互。
- 今日轨迹展示。
- 能量风向标展示。
