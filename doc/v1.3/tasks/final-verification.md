## 输出

## 建议验证命令

```bash
npm run test
npm run typecheck
npm run lint
npm run build
npm run dev
```

## 验收标准

v1.3 完成时必须满足：

- 首次启动扫描仍然可用。
- 完成扫描后进入 LifeOS 启动面板。
- 个人生态系统可进入并保持 v1.2 闭环。
- 能量管理系统可进入。
- 能量管理系统形成记录、保存、回顾的最小闭环。
- 其他四个子系统没有被提前实现。
- 用户界面没有 JSON 导出入口。
- 用户界面没有 Markdown 导出入口。
- 用户界面没有完整个人说明书编辑入口。
- 用户界面没有诊断、AI 分析、健康建议、医疗建议、任务规划、恢复建议或评分压力。
- 重置后回到首次启动状态。
- 本地优先原则保持不变。
