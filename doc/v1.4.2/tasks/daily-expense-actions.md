# 日常开销池业务动作任务

## 目标

实现日常开销池的确定性业务动作：从钱包容器划入、消费扣款、删除消费流水并回滚开销池余额。

## 最小可执行任务

划款时来源钱包容器余额减少、开销池余额增加；消费时开销池余额减少并生成消费流水；删除消费流水时开销池余额加回，钱包容器不变。

## 输入

- `DailyExpensePool`
- `DailyExpenseEntry`
- `WalletContainer`
- 钱包容器保存函数。
- 日常开销池保存函数。
- 消费流水保存和删除函数。
- `doc/v1.4.2/design.md`

## 输出

- 划款动作。
- 消费动作。
- 删除消费流水回滚动作。
- 金额校验。
- 备注校验。
- 划款来源状态判断。
- 对应单元测试。
