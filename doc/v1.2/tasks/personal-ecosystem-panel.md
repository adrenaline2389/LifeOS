# 个人生态系统页面任务

## 目标

实现个人生态系统的三个面板：当前生态状态、今日生态轨迹、生活晴雨表。

## 最小可执行任务

用户可以进入个人生态系统，选择性记录某一个维度，并看到当前状态、今日轨迹和生活晴雨表更新。

## 输入

- 生态维度配置。
- 生态观察点存储函数。
- 生态聚合函数。
- 复古 UI 系统组件。
- `doc/v1.2/design.md`

## 输出

- 个人生态系统页面 UI。
- 选择性记录交互。
- 今日轨迹展示。
- 生活晴雨表展示。
