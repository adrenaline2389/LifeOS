## 验收标准

v1.4.3 完成时必须满足：

- 首次启动扫描仍然可用。
- 完成扫描后进入 LifeOS 启动面板。
- 个人生态系统可进入并保持 v1.2 闭环。
- 能量管理系统可进入并保持 v1.3 闭环。
- 财务管理系统可进入。
- 财务管理系统保持“我的钱包”的本地余额快照闭环。
- 财务管理系统保持“收入来源”的本地手动入账闭环。
- 财务管理系统保持“日常开销池”的本地手动划款和消费扣减闭环。
- 财务管理系统新增“财富流动日志”的本地追加式事件记录闭环。
- 日常开销池用户侧不再展示消费流水列表。
- 认知管理系统、人生目标管理系统和人际关系管理系统没有被提前实现。
- 用户界面没有 JSON 导出入口。
- 用户界面没有 Markdown 导出入口。
- 用户界面没有完整个人说明书编辑入口。
- 用户界面没有预算警报、超支惩罚、消费建议、投资建议、财务诊断、银行连接、支付连接或 AI 分析。
- 不追溯旧数据。
- 不用日志重算余额。
- 不做完整账本事件源重构。
- 重置后回到首次启动状态。
- 本地优先原则保持不变。
