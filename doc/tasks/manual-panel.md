# 个人说明书面板模块任务

## 目标

展示 onboarding 后生成的个人说明书控制面板，并提供完整说明书入口、导出和重置操作。

## 最小可执行任务

用户完成 onboarding 后，可以看到自我清晰度、已识别参数、待验证观察、建议开启的子系统和可编辑说明书入口。

## 输入

- `ManualProfile`
- 复古 UI 系统模块
- 导出与重置模块

## 输出

- 个人说明书控制面板 UI
