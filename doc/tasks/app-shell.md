# App Shell 模块任务

## 目标

实现 LifeOS v1.0 的应用外壳，负责首次启动判断、页面级布局和高层流程切换。

## 最小可执行任务

当用户打开 app 时，App Shell 能根据本地是否已有个人说明书档案，决定展示 onboarding 流程或个人说明书控制面板。

## 输入

- 本地数据模块提供的读取结果
- Onboarding 模块
- 个人说明书面板模块
- 复古 UI 系统模块

## 输出

- 首次启动状态下展示 onboarding
- 已完成 onboarding 状态下展示个人说明书控制面板
- 重置后回到首次启动状态
