# 子系统推荐模块任务

## 目标

根据 onboarding 回答信号生成最多 2 个建议开启的子系统，并保留推荐来源。

## 最小可执行任务

实现一个确定性的推荐函数。它只负责推荐，不负责强制用户选择，也不创建真实子系统数据。

## 输入

- `OnboardingAnswerRecord`
- 问题 schema 中的 signal tags

## 输出

- `SuggestedSubsystem[]`
