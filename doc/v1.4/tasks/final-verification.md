## 验收标准

v1.4 完成时必须满足：

- 首次启动扫描仍然可用。
- 完成扫描后进入 LifeOS 启动面板。
- 个人生态系统可进入并保持 v1.2 闭环。
- 能量管理系统可进入并保持 v1.3 闭环。
- 财务管理系统可进入。
- 财务管理系统形成“我的钱包”的本地余额快照闭环。
- 认知管理系统、人生目标管理系统和人际关系管理系统没有被提前实现。
- 用户界面没有 JSON 导出入口。
- 用户界面没有 Markdown 导出入口。
- 用户界面没有完整个人说明书编辑入口。
- 用户界面没有预算警报、超支惩罚、投资建议、财务诊断、银行连接、支付连接或 AI 分析。
- 重置后回到首次启动状态。
- 本地优先原则保持不变。
