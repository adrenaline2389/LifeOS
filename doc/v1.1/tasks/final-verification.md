# 全量回归与验收任务

## 目标

在所有 v1.1 模块合并后做最终验收。

## 最小可执行任务

确认 v1.1 符合 `doc/v1.1/design.md`，并通过测试、类型检查、lint 和本地浏览器流程验证。

## 输入

- 完整 v1.1 实现
- `doc/v1.1/design.md`
- `doc/v1.1/tasks/progress.md`

## 输出

- 全量验证结果
- 更新后的任务进度

## 建议验证命令

```bash
npm run test
npm run typecheck
npm run lint
npm run build
npm run dev
```

## 验收标准

v1.1 完成时必须满足：

- 页面主标题不再是“个人说明书控制面板”。
- 主界面展示六个子系统，而不是说明书章节。
- 推荐结果最多 2 个。
- 用户界面没有 JSON 导出入口。
- 用户界面没有 Markdown 导出入口。
- 用户界面没有完整个人说明书编辑入口。
- 重置后回到首次启动状态。
- 本地优先原则保持不变。
